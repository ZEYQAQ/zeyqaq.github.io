发币流程：
1、安装homebrew，你可以用下面的命令来验证你是否安装

```
eval "$(/opt/homebrew/bin/brew shellenv)"
brew --version
```

2、安装solana-cli和metaplex（mplx）

```
curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
npm install -g @metaplex-foundation/cli
```

3、添加至环境变量

```
export PATH="$HOME/.local/share/solana/install/active_release/bin:$PATH"
echo 'export PATH="$HOME/.local/share/solana/install/active_release/bin:$PATH"' >> ~/.zprofile
source ~/.zprofile
```

4、生成一个私钥和公钥地址

```
zeyqaq@zeyqaqdeMac-mini ~ % solana-keygen new

NOTE! This passphrase improves security of the recovery seed phrase NOT the
keypair file itself, which is stored as insecure plain text

========================================================================
pubkey: D4CYZtn5JYeH7VVVF91ASMnJCeLCTVt8zaxJoYxQxMu3
========================================================================
Save this seed phrase and your BIP39 passphrase to recover your new keypair:
later ethics romance wet adapt cancel floor lock bring ask motor poverty
========================================================================
zeyqaq@zeyqaqdeMac-mini ~ % 
```

5、切换至主网

```
solana config set --url https://api.mainnet-beta.solana.com
solana config get
```

6、测试网上使用

```
solana config set -udv
```

7、请求空投

```
solana airdrop 5
#如遇到错误，请访问：https://faucet.solana.com/，需使用github鉴权登陆
```

8、验证自己的账户余额并切换钱包

```
solana balance
mplx config wallets add dev1 ~/.config/solana/id.json
mplx config wallets set dev1
```

9、铸币

```
cd /Users/zeyqaq/ZEY/

mplx toolbox token create \
  --name "ZEY" \
  --symbol "ZEY" \
  --description "ZEY official token" \
  --image ./zeycn-icon-512.png \
  --decimals 0 \
  --mint-amount 100
```
